# BUMP Archive

这里是《大邮数学集》（*Bigdu[^1] University Mathematics Problemset*）的归档仓库。

读者可以在右侧 Releases 栏中找到最新版本。

本仓库不包含《大邮数学集》的源码；实际上，《大邮数学集》目前并不开放源码和社区 PR。如果您想参与《大邮数学集》的建设，请参见 [Contribution](#Contribution)。

## Community

- GitHub 归档仓库（本仓库）。
  - 在此**获取最新版本**。
  - 通过 Issues 区或其他方式联系主编（仓库所有者）。

  - 链接带有本文件的版本号信息。因此，请您**不要转发**该链接，并确保本文件是当前的最新版本，否则将被我们的工作流自动标记为无效问卷。

## Contribution

若您想提供题目、校对、维护、排版、宣传或法律支持，抑或是想帮我们给积压的题目做做答案（可借助 AI / LLM，但您需对正确性负责），请参照 [Community](#community) 一节联系我们。衷心感谢您的**无偿**贡献，您将有权把自己加入贡献者列表中。

## Roadmap

本文档将持续收集反馈、套题，并发布新版本；也乐于接受更多数学课程的资料。

相信大家都能看出编辑工作量的巨大，因此我们很遗憾地暂时无力提供答案的详细解析。

研究生数学课程的题目来源较少且罕有免费分享，因此数量不多。

请不要在各种渠道询问或催促《离散数学》课程的题目。其一，编者认为该课并非属于数学类课程，而应属于计算机科学基础课程；其二，该课程是学校开课最为杂乱的课程（没有之一），几乎所有工科专业都有开设，但每个学院都有各自的一套甚至多套内容差异极大的课程和题目。这导致每套题的整理工作的服务对象相对较窄，我们也无力收集整理。该课程可能在将来被纳入其他项目中。

若没有更多朋友协助，我们将可能在较近的未来（也许一年内？）启动《大邮死灵疤》等新的项目，敬请期待。

本书有计划在一段时间内将排版模板的源码独立公开，但尚未计划开放内容的源码。

[^1]: 我们知道大都是 Dadu 或者 the Great Capital，叫作 Bigdu 只是想凑出一个 B 来。
