# BUMP Archive

这里是《大邮数学集》（*Bigdu[^1] University Mathematics Problemset*）的归档仓库。

读者可以在右侧 Releases 栏中找到需要的版本，当然也包括最新版本。

本仓库不包含《大邮数学集》的源码；实际上，《大邮数学集》目前并不开放源码和社区 PR。如果您想参与《大邮数学集》的建设，请参见 [Contribution](#Contribution)。

## Contribution

若您想提供题目、校对、维护、排版、宣传或法务支持，抑或是想帮我们给**积压的大量题目**做做答案，请参照 [Community](#community) 一节联系我们。为保持材料的一致性和使用体验，若提供出自课程组之手的试题原件（含扫描版与手抄版），谢绝贡献者署名；若提供回忆版，请保证该内容非自他人手中剽窃，完整性和准确性较高，您必须监督排版全过程，必须署名并提供联系方式。衷心感谢您的**无偿**贡献，您将有权把自己加入**贡献者列表**中。

## Roadmap

本文档将持续收集反馈、套题，并发布新版本；也乐于接受更多数学课程的材料。

相信大家都能看出编辑工作量的巨大，因此我们很遗憾地暂时无力提供答案的详细解析；在不可预见的遥远未来，本材料将有可能开放源码或开放社区 PR。

我们已经在内部版本纳入了数门研究生数学课程的题目，但因材料来源有限、罕有人愿意免费分享、缺少答案等原因暂未在公开版本中出现。

请不要在各种渠道询问或催促《离散数学》课程的题目。其一，编者认为该课并非属于数学类课程，而应属于计算机科学基础课程；其二，该课程是学校开课最为杂乱的课程（没有之一），所有专业都有开设，但每个学院都有各自的一套甚至多套内容差异极大的课程和题目。这导致每套题的整理工作的服务对象相对较窄，我们也无力收集整理。该课程可能在将来被纳入其他材料项目中。

若没有更多朋友协助，我们将可能在较近的未来（也许一年后？）启动《大邮死灵疤》等新的材料项目，敬请期待。

[^1]: 我们知道大都是 Dadu 或者 the Great Capital，叫作 Bigdu 只是想凑出一个 B 来。
